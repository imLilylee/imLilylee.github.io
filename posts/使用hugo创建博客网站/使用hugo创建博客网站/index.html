使用hugo创建博客网站

参考文档

  • https://www.gohugo.org/
  • https://toha-guides.netlify.app/posts/getting-started/prepare-site/

主题

  • toha

步骤

1. 安装hugo

  • 打开 https://github.com/gohugoio/hugo/releases ，选择hugo_extended_0.95.0_Windows-64bit.zip 下载

  • 解压 hugo_extended_0.95.0_Windows-64bit.zip

  • 将hugo.exe所在目录添加到系统环境变量path中

  • 在命令行输入 hugo version ,如果输出版本即安装成功！

2. 生成站点

  • 创建一个目录，用于存放站点内容。

  • 进入到目录下，输入以下命令： hugo new site ./ -f=yaml --force

    这样就在目录里生成了初始站点。站点目录结构：

    archetypes/
content/
layouts/
static/
config.yaml

3. 添加主题

git submodule add https://github.com/hugo-toha/toha.git themes/toha

4. 本地运行站点

hugo server --theme=toha --buildDrafts

此时，打开 http://localhost:1313 就可以看到网站了。

5. 配置网站

此处暂时省略

6. 添加posts

hugo new posts/xxx/xxx.md

7. 部署到远程Github Page

  • 在github上新建仓库，并以“你的github用户名.github.io”命名仓库
  • 在本地运行hugo --theme=hyde --baseUrl="http://你的github用户名.github.io/"生成静态页面,这些都会生成到 public 目录。 （注意，以上命令并不会生成草稿页面，如果未生成任何文章，请去掉post文章头部的 draft=true 再重新生成。）
  • 将public目录下的所有文件push到刚创建的 github仓库 中
$ cd public
$ git init
$ git remote add origin https://github.com/你的github用户名/你的github用户名.github.io.git
$ git add .
$ git commit -m "xxx"
$ git push -u origin master
